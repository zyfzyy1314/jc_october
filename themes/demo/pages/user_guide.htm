>> User Guide

Welcome to the Algoal365 user guide! The Algoal365 website is a user friendly application that can take a short time to explore and understand. While this user guide only covers a fraction of the website, it will give you a great head-start in understanding the site's functionality and underlying rationale.

Algoal365 is entirely focused on curating financing-market information for traders. The nature of this information is primarily real-time (i.e., what's happening right now), and the scope ranges from economic data to analysis from members.

Algoal365 hosts three main functions: Calendar, News, and Add to website. There's a link to each of these products in the header of every page.

CALENDAR

1. Last / Next Week: To synchronize the calendar records with your selected week.

2. Current date: You can browse current date records by clicking the day of the week.

3. This Week: To synchronize the calendar records with your week of local time,

4. Filter: Expend the query options for calendar records.

5. Chart: Throughout the calendar you will see chart icons to the right of record and it will expend the chart of history data. s

6. Context: Click on the yellow down arrow associated with the record, it will extend the context panel within this record.

NEWS

The NEWS page is a real-time updating page and you can get the latest marketing news. Also, you could click the link to brows the context from the source website.

ADD TO WEBSITE

A Widget is a tool that allows you to perform a function or access different services. We offer a great selection of Widgets that allow you to access our provided calendar and new.
